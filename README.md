## ✨ Features

### 🔐 **Authentication System**
- Secure login with admin credentials
- Password recovery with security questions
- Default login: Username: `admin`, Password: `admin`

### 👨‍🎓 **Student Management**
- Add, update, and delete student records
- Store student details (ID, Roll No, Name, Department)
- View all students in tabular format
- Sample data included for testing

### 📸 **Face Recognition**
- Real-time face detection using webcam
- LBPH (Local Binary Patterns Histograms) algorithm
- Automatic attendance marking on face recognition
- Confidence level display (77% threshold)
- Unknown face detection

### 📊 **Attendance Management**
- Auto-mark attendance with date/time stamps
- Manual attendance entry option
- Import/Export attendance data in CSV format
- View, update, and delete attendance records
- Dual-panel attendance view

### 🗄️ **Database Integration**
- MySQL database for persistent storage
- Automatic table creation on first run
- Sample data insertion
- Secure connection handling

### 🖥️ **User Interface**
- Professional Tkinter GUI
- Responsive button panels
- Real-time clock display
- Image-based navigation buttons
- Intuitive layout
